# Personal Web Syah Putra Nugraha

Selamat datang di repository **Personal Web Syah Putra Nugraha**. Ini adalah pusat pengembangan website pribadi yang dibangun dengan pendekatan modern, terstruktur, dan profesional.

## 📂 Struktur Project

Repository ini menggunakan struktur 2 level untuk memisahkan antara manajemen project dan implementasi teknis:

- **`README.md`**: Dokumentasi utama repository (file ini).
- **`cutback-master/`**: Dokumentasi internal, perencanaan, roadmap, task list, dan catatan pengembangan.
- **`webstrip/`**: Source code utama website (Aplikasi Node.js/Vite).

## 🚀 Pengembangan & Deployment

Project ini dirancang untuk kemudahan deployment ke berbagai platform:
- [x] **Vercel**
- [x] **Netlify**
- [x] **GitHub Pages**

## 🛠️ Tech Stack (Rencana)

- **Core**: HTML, CSS, JavaScript (Node.js)
- **Framework/Tooling**: Vite (untuk build tool yang cepat)
- **Deployment**: CI/CD via GitHub Actions
