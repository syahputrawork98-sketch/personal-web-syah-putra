# BATCH TEMPLATE

Gunakan template ini untuk setiap pembuatan catatan batch baru di folder history.

```markdown
# Batch XX — Nama Fitur atau Perubahan

Tujuan:
- [Deskripsi tujuan utama batch ini]

Scope kerja:
1. [Langkah 1]
2. [Langkah 2]

Perubahan:
- [File yang dibuat]
- [File yang diubah]
- [Folder/file yang dihapus]

Catatan teknis:
- [Penjelasan teknis atau kendala yang dihadapi]
```
