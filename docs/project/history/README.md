# History

Folder ini berisi dokumen riwayat dan status aktif project:
- `CURRENT_STATUS.md` sebagai **satu-satunya file status aktif** (termasuk status HOLD dan deployment notes).
- `BATCH_TEMPLATE.md` sebagai panduan format pencatatan batch.
- `BATCH_01_TO_10.md`, `BATCH_11_TO_20.md`, `BATCH_21_TO_30.md` sebagai **arsip** riwayat batch yang sudah selesai dikerjakan.
- Kode `-CP` (misal Batch 26-CP) digunakan khusus untuk **checkpoint dokumentasi**. Checkpoint merapikan history dan catatan frontend/backend/database sesuai pekerjaan sebelumnya.

*Catatan: Referensi lama dari `docs/01-batch-history/` dan `docs/02-project-status/` telah sepenuhnya dimigrasikan ke sini pada Batch 26C. Selain itu, file `HOLD_ITEMS.md` dan `DEPLOYMENT_NOTES.md` telah digabung ke `CURRENT_STATUS.md` pada Batch 26I.*
