# Workflow

Folder ini berisi sistem kerja utama project **PW Personal Web**.

Dokumen utama:

- `WORKING_SYSTEM.md` — sistem kerja project, role, alur batch, aturan eksekusi, dan aturan review.

## Fungsi Folder

Folder ini digunakan untuk memahami:
- Peran user, Roomchat 00, Roomchat 01, Anti-Gravity IDE, dan Gemini Anti-Gravity.
- Alur kerja batch dari diskusi sampai commit.
- Aturan eksekusi agar perubahan tetap kecil dan aman.
- Aturan review sebelum lanjut ke batch berikutnya.

## Catatan Migrasi
