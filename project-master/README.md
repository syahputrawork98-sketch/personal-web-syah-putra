# Project Control Center

Welcome to the `project-master` directory. This is the single source of truth for the portfolio project's roadmap, checklists, and system planning.

## Workflow

The workflow of the project follows a strict top-down structure:
1. **`project-master`**: First, define the system architecture, API contracts, and roadmap here.
2. **`backend`**: Next, implement the models, controllers, and APIs according to the planned contract.
3. **`webstrip` (Frontend)**: Finally, build the user interfaces and connect them to the backend API.
