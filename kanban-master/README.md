# Project Kanban Master

`kanban-master` adalah pusat kontrol utama untuk project portfolio. Semua dokumentasi perencanaan, checklist, contract API, dan roadmap harus dipusatkan di sini.

## Alur Kerja
- `kanban-master` → `backend` → `webstrip`
- Backend dan webstrip tetap merupakan folder kode utama.
- Setiap fitur baru harus dicatat terlebih dahulu di `kanban-master` (sebagai perencanaan/checklist) sebelum diimplementasikan di kode.
