# Project Kanban Master

`kanban-master` adalah pusat kontrol utama untuk project portfolio ini. Semua dokumentasi perencanaan, checklist fitur, dan status implementasi harus dipusatkan di sini.

## Alur Kerja
- **`kanban-master`** → **`backend`** → **`webstrip`**
- Folder `backend` dan `webstrip` tetap merupakan folder kode utama untuk fungsionalitas aplikasi.
- Setiap fitur baru harus dicatat dan direncanakan terlebih dahulu di `kanban-master` sebelum mulai diimplementasikan di kode.
